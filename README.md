## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

* python
  ```python
  pip install -r requirements.txt
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/josephAttia/coding-game-dev.git
   ```
2. Install Python Libraries
   ```python
    pip install -r requirements.txt
  ```
